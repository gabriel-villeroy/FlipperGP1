# Flipper The Link To The Past



Mon flipper se contrôle à l'aide des touches directionnelles gauche et droite pour les paddles et avec la touche espace pour changer de "mode". La touche shift gauche permet également de faire apparaitre une nouvelle balle lorsque cela est nécessaire. 



Un menu Pause était prévu et existe actuellement dans le jeu, il a simplement arrêté de fonctionner 24h avant le rendu (il est toujours dans le projet et dans le code cela dit). Mis à part ça et le shooter de la balle que j'ai essayé de reproduire sans succès, pas de difficultés particulières à souligner.



Je suis ouvert à toute consultation / retours sur mon code (bien que je sache que cela ne fait pas partie de la notation).



Ah et le nom est une ref au Zelda ou on peut switch entre deux mondes parallèles :)
